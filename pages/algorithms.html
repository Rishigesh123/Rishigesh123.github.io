Supervised Learning Algorithms

Linear Regression

Type: Regression

A simple algorithm that models the relationship between a dependent variable and one or more independent variables by fitting a linear equation to observed data.

Real-time Use: Predicting house prices based on size, location, and age.

Logistic Regression

Type: Classification

Used to estimate discrete values (binary values like 0/1, yes/no, true/false) based on a given set of independent variable(s).

Real-time Use: Spam detection in emails (Spam or Not Spam).

Decision Tree

Type: Classification & Regression

A tree-structured classifier, where internal nodes represent the features of a dataset, branches represent the decision rules, and each leaf node represents the outcome.

Real-time Use: Loan approval prediction (approve/deny based on credit score, income, etc.).

Support Vector Machine (SVM)

Type: Classification

Finds a hyperplane in an N-dimensional space that distinctly classifies the data points.

Real-time Use: Face detection, handwriting recognition.

Unsupervised Learning Algorithms

K-Means Clustering

Type: Clustering

An iterative algorithm that partitions the dataset into K pre-defined distinct non-overlapping subgroups (clusters) where each data point belongs to only one group.

Real-time Use: Customer segmentation for marketing (grouping customers by purchasing behavior).

Principal Component Analysis (PCA)

Type: Dimensionality Reduction

A statistical procedure that uses an orthogonal transformation to convert a set of observations of possibly correlated variables into a set of values of linearly uncorrelated variables called principal components.

Real-time Use: Image compression, visualizing high-dimensional data.
