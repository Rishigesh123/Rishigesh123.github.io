Linear Regression

Predicting values with surgical precision.

REGRESS

The Intuition

Linear Regression is the "Hello World" of Machine Learning. It's a supervised learning algorithm used to predict a continuous numerical value based on one or more input features.

Imagine you have a scatter plot of house sizes vs. prices. Linear regression finds the single best straight line that passes through those points with the minimum total error.

The Hypothesis

The core of the algorithm is the straight-line equation:

Y = β₀ + β₁X + ε
  • Y: The target value we want to predict (e.g., Price).
  • X: The input feature (e.g., Size).
  • β₀ (Intercept): Where the line crosses the Y-axis.
  • β₁ (Slope): The weight or importance of the input feature.

How It Works

  1. Loss Calculation: We measure the distance between the actual data points and our line (the "errors").
  2. Minimize Error: Ideally, we use Mean Squared Error (MSE) to penalize large errors more heavily.
  3. Optimization: We use an optimizer (like Gradient Descent) to nudge β₀ and β₁ until the error is as small as possible.

Cost Function (MSE)

J(θ) = (1/2n) * Σ (Predictedᵢ - Actualᵢ)²

ALGORITHM TYPE

Supervised (Regression)

TIME COMPLEXITY

O(n * p² + p³)

n: samples, p: features

STRENGTHS

  • Mathematically robust
  • Extremely fast
  • High interpretability

WEAKNESSES

  • Assumes linearity
  • Sensitive to outliers
  • Prone to underfitting
