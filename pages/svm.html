Support Vector Machine (SVM)

Drawing the ultimate boundary between data classes.

BOUNDARY

The Intuition

SVM is a supervised machine learning algorithm used primarily for classification. It works by finding the "hyperplane" that best separates data into different classes.

Imagine a battlefield with two armies. SVM doesn't just draw a line between them; it finds the widest possible "No Man's Land" (margin) to ensure they are as far apart as possible.

The Margin

The goal of SVM is to maximize the distance between the hyperplane and the nearest data points from either class. These critical points are called Support Vectors.

By maximizing this margin, the algorithm becomes more robust and less prone to misclassifying new, unseen data.

The Kernel Trick

What if the data can't be separated by a straight line? SVM uses a "Kernel" to project the data into a higher dimension where a linear separation is possible. Common kernels include RBF (Radial Basis Function) and Polynomial.

Hard vs. Soft Margin

A "Hard Margin" allows zero mistakes but can fail on noisy data. A "Soft Margin" uses a penalty parameter (C) to allow some misclassifications in exchange for a wider, more generalizable boundary.

ALGORITHM TYPE

Supervised (Classification)

STRENGTHS

  • High accuracy in high dimensions
  • Effective with small datasets
  • Versatile via Kernel trick

WEAKNESSES

  • Slow on very large datasets
  • Sensitive to feature scaling
  • Hard to interpret (Black box)
